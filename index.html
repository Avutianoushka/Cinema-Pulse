AWS Deployment Guide

Complete step-by-step instructions to deploy CinemaPulse on AWS EC2 with RDS MySQL

Project Structure 

cinemapulse/
├── app.py                 # Flask application (main entry point)
├── requirements.txt       # Python dependencies
├── deploy.sh             # EC2 deployment script
├── setup_rds.sql         # MySQL database setup
├── .env                  # Environment variables (create on EC2)
└── templates/
    ├── base.html         # Base template with navigation
    ├── index.html        # Home page (movie listings)
    ├── movie_detail.html # Movie details + feedback form
    ├── thank_you.html    # Feedback confirmation
    ├── analytics.html    # Analytics dashboard
    └── error.html        # Error page
1

Create EC2 Instance

AWS Console → EC2 → Launch Instance

  • Name: cinemapulse-server
  • AMI: Amazon Linux 2023 or Ubuntu 22.04
  • Instance Type: t2.micro (Free Tier)
  • Key Pair: Create new or select existing
  • Storage: 8 GB gp2

Security Group Rules:

  • SSH (22) - Your IP
  • HTTP (80) - Anywhere
  • Custom TCP (5000) - Anywhere
2

Create RDS MySQL Instance

AWS Console → RDS → Create Database

  • Engine: MySQL 8.0
  • Template: Free Tier
  • DB Instance ID: cinemapulse-db
  • Master Username: admin
  • Master Password: Your secure password
  • Instance: db.t3.micro
  • Public Access: Yes (for initial setup)

RDS Security Group:

Add inbound rule: MySQL/Aurora (3306) from EC2 Security Group

3

SSH & Deploy Application

# Connect to EC2
ssh -i your-key.pem ec2-user@YOUR-EC2-PUBLIC-IP

# Update system
sudo yum update -y

# Install dependencies
sudo yum install python3 python3-pip git -y

# Clone project (or upload via SCP)
git clone https://github.com/yourusername/cinemapulse.git
cd cinemapulse

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install Python packages
pip install -r requirements.txt
4

Configure Database

# Connect to RDS and setup database
mysql -h YOUR-RDS-ENDPOINT.rds.amazonaws.com -u admin -p

# Run the setup script
mysql> source setup_rds.sql

# Verify
mysql> USE cinemapulse;
mysql> SHOW TABLES;
mysql> SELECT COUNT(*) FROM movies;
5

Set Environment Variables

# Create .env file
nano .env

# Add these variables:
FLASK_APP=app.py
FLASK_ENV=production
SECRET_KEY=your-super-secret-random-key

DB_HOST=your-rds-endpoint.rds.amazonaws.com
DB_PORT=3306
DB_USER=admin
DB_PASSWORD=your-rds-password
DB_NAME=cinemapulse
6

Run Application

Development Mode

python app.py

# Access at:
http://YOUR-EC2-IP:5000

Production Mode

gunicorn -w 4 -b 0.0.0.0:5000 app:app

# Or use the deploy script:
./deploy.sh
7

Enable RDS in app.py

Uncomment the MySQL configuration section in app.py:

import mysql.connector
from mysql.connector import pooling
import os

DB_CONFIG = {
    'host': os.environ.get('DB_HOST'),
    'user': os.environ.get('DB_USER'),
    'password': os.environ.get('DB_PASSWORD'),
    'database': os.environ.get('DB_NAME'),
    'pool_name': 'cinemapulse_pool',
    'pool_size': 5
}

def get_db_connection():
    connection_pool = mysql.connector.pooling.MySQLConnectionPool(**DB_CONFIG)
    return connection_pool.get_connection()
